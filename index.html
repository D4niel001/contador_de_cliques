Contador Numérico

000000
  • O número pode estar entre -99999 e 999999.
  • Pode ser usado para contar pessoas, marcar ou contar cliques do mouse.
  • O número é editável.
  • Basta clicar para contar. Clique no botão para adicionar um e clique no botão para menos um. Botões do teclado também estão disponíveis. Pressione seta para cima para adicionar e seta para baixo para subtrair.
  • Este é um aplicativo da web online. Nenhuma instalação é necessária.
